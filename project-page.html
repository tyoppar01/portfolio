Deception Detection System

Developed a machine learning–powered web application with a Gradio interface that analyzes facial expressions from uploaded videos to detect deceptive behavior. The system extracts multimodal features—primarily Facial Action Units (FAUs) using the OpenFace model, and predicts the likelihood of deception based on trained models. The output includes both the deception probability and the key FAUs contributing to the prediction.

  • Gradio
  • OpenFace
  • Scikit-Learn
